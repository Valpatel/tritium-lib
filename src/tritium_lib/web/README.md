# Web — Cyberpunk Theme Engine

Shared HTML/CSS theme engine and dashboard components for the Tritium ecosystem. Provides reusable cyberpunk-styled UI elements that can be used by any Tritium service (fleet server, test reports, standalone tools).

## Files

| File | Purpose |
|------|---------|
| `theme.py` | Color palette constants (cyan, magenta, green, yellow) |
| `components.py` | Reusable HTML component generators (cards, tables, bars) |
| `templates.py` | Full page templates with cyberpunk styling |
| `dashboard.py` | Dashboard layout generator |

## Colors

- Cyan: `#00f0ff`
- Magenta: `#ff2a6d`
- Green: `#05ffa1`
- Yellow: `#fcee0a`
- Background: `#0a0a0f`
