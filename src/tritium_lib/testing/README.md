# Tritium-Lib Testing

Shared test utilities used across tritium-sc and tritium-edge test suites.

## Key Files

- `visual.py` — VisualCheck: OpenCV-based visual regression validation
- `flicker.py` — Flicker detection for UI stability checks
- `device.py` — ESP32 device automation (serial, flash, monitor)
- `runner.py` — Test runner utilities and report generation

## Related

- SC tests: `tritium-sc/tests/`
- Edge tests: build verification via PlatformIO
